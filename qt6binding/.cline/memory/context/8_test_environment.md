# テスト環境

## 6.1 テストフレームワーク
- 単体テスト
- 統合テスト
- メモリリークテスト

## 6.2 テスト実行環境
```bash
# テストのビルドと実行
cmake --build . --target test
```

## 6.3 テストカバレッジ
- ライン: 80%以上
- ブランチ: 70%以上
- 関数: 90%以上

## 6.4 テスト種別
1. 単体テスト
   - クラスごとの機能テスト
   - エッジケースの検証
   - エラー処理の確認

2. 統合テスト
   - コンポーネント間の連携
   - シグナル/スロットの動作
   - リソース管理の検証

3. パフォーマンステスト
   - メモリ使用量
   - 実行速度
   - リソース解放
